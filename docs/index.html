gemini-cli-scanner v3.3.1

Discover patterns in your Gemini CLI and Claude Code environments.

📊 Maturity Scoring 🔍 Pattern Detection ⚡ Skill Generation

Extract tribal knowledge from how you actually use AI coding tools. The scanner reads your conversations, identifies repeating patterns in prompts, tool calls, and workflows, then generates production-ready SKILL.md files you can use yourself or share with your team. Get a maturity report scoring your environment across 8 categories with actionable, doc-linked recommendations.

npx gemini-cli-scanner
Or install as a Gemini CLI extension: gemini extensions install ↗

How It Works

💬 Scan Conversations

Reads your chat history, tool usage, config files, and multi-tool ecosystem. All processing is local — no data leaves your machine.

🤖 AI Pattern Detection

A two-stage Gemini pipeline identifies repeating workflows from your real usage data and generates reusable skill templates.

📋 Maturity Report

Scores your environment 0–115 across 8 categories — from policy hygiene to MCP governance — with actionable recommendations.
